# MAP

  Main folders for the Gramado OS

## Bootloader

```
boot/arch/x86/bsp/
```

## Kernel

```
kernel/             (Base kernel)
modules/dungeon/    (Main kernel module)
```

## Init process

moon/               (The init process)


## Native UI

```
hunter/  - Compositor (display server)
ifarmer/ - Client-side GUI applications
```

## Posix-like commands

```
np/cmds/
```
