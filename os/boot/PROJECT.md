# boot

Project: boot

# Description

```
    This is the boot loader for the system.
    The main folder for this project is: boot/
    Main programs for x86:
    + boot/arch/bsp/vd      - The MBR and the virtual disk creation
    + boot/arch/bsp/bm/     - 16/32 Boot manager
    + boot/arch/bsp/blgram/ - 32bit Boot loader
```

# Environment

```
    Ring 0
```
