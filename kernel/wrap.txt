

Sketching out a namespace taxonomy

_core     - Core kernel logic.
_wrappers - Glue functions.
_DDINIT - Device Driver Initialization
_devmgr - Device manager.
-sched - Scheduler functions.
_ts    - Task switch routines.
_mm    - Memory management.
_ke    - Kernel executive (execution unit)
_net     - Networking subsystem.
_network - Networking subsystem.
_fs    - Filesystem support.
_wink  - wink sybsystem. (Wrappers for keyboard, mouse and display)
_ksys  - Kernel‑internal syscalls. Used when the kernel itself invokes syscall‑like routines (internal API).
_sys   - System call wrappers
