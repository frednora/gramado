# per - Peripherals

 A place for Peripherals support.

## Folders

```
  * blkdev   - Block devices.
  * bus      - A place for pci, usb stuff.
  * chardev/ - Char devices.
  * fs       - Filesystems
  * mm       - Memory management.
  * net      - Network management.
             - Network initialization, socket, domains and protocols.
  * netdev/  - Network device drivers.
```
