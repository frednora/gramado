# include - Main header for the kernel 

```
config/      - Configuration files.
 + config.h  - Main configuration file.
 + product.h - Product information. (name, type).
 + u.h       - os name, version, release, machine. node, domain, version, username. 
 + utsname.h - 'UNIX Time-Sharing' standard variables.
 + version.h - Version information.

kernel.h - Main header for the kernel.
```
