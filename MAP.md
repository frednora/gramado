# MAP

  Main folders for the Gramado OS

## Bootloader

```
boot/arch/x86/bsp/
```

## Kernel

```
kernel/             (Base kernel)
modules/dungeon/    (Main kernel module)
moon/               (The init process)
```


## Native UI

```
farmer/ - Client-side GUI applications
hunter/ - Compositor (display server)
```

## Posix-like commands

```
np/cmds/
```
